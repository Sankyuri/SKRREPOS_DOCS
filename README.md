# Naming Rules
* Alphabets and numbers are usable.
* Delimiter is underscore '_'.
* First section:
  1. The signature "SKR".
  1. The class expressed with 2 chars e.g. "PG".
  1. The serial number with 3 digits.
* Second section:
  1. The repository name. Words can be separate by '_'.
